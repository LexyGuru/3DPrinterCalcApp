## v3.0.0 (2025) - 🔒 Шифрування Даних Клієнтів & Відповідність GDPR

### 🔒 Шифрування Даних Клієнтів
- **Шифрування AES-256-GCM** - Зашифроване зберігання даних клієнтів з використанням галузевого стандартного алгоритму AES-256-GCM
- **Хешування паролів PBKDF2** - Безпечне зберігання паролів з використанням алгоритму PBKDF2 (100 000 ітерацій, SHA-256)
- **Зберігання в окремому файлі** - Зашифровані дані клієнтів зберігаються в окремому файлі `customers.json`
- **Управління паролями в пам'яті** - Паролі зберігаються лише в пам'яті та видаляються при закритті додатку
- **Інтеграція пароля додатку** - Опціонально: пароль захисту додатку також може використовуватися для шифрування
- **Система запиту пароля** - Інтелектуальний запит пароля (не відображається на екрані завантаження, після привітального повідомлення)
- **Захист цілісності даних** - Зашифровані дані захищені від випадкового видалення

### ✅ Захист Даних у Відповідності з GDPR/ЄС
- **Відповідність**: Додаток обробляє дані клієнтів відповідно до GDPR (Загальний регламент щодо захисту даних) та правилами захисту даних ЄС
- **Галузеве стандартне шифрування**: Використання алгоритму AES-256-GCM (відповідає рекомендаціям ЄС)
- **Безпечне управління паролями**: Алгоритм хешування PBKDF2 (рекомендовано NIST)
- **Мінімальний збір даних**: Зберігає лише необхідні дані клієнтів, які вимагає додаток
- **Зберігання даних**: Користувач має повний контроль над зберіганням та видаленням даних
- **Контроль доступу**: Доступ, захищений паролем, до даних клієнтів

### 🎨 Покращення UI/UX
- **Модальне вікно активації шифрування** - Нове модальне діалогове вікно для активації шифрування з опцією пароля додатку
- **Розширення ConfirmDialog** - Підтримка користувацького вмісту для модальних компонентів
- **Таймінг запиту пароля** - Інтелектуальне відображення (не на екрані завантаження)
- **Інтеграція налаштувань** - Налаштування шифрування на вкладці Безпека

### 🔧 Технічні Покращення
- **Модуль шифрування backend** - Шифрування, реалізоване на Rust (`src-tauri/src/encryption.rs`)
- **Утиліти шифрування frontend** - Допоміжні функції TypeScript для управління шифруванням
- **Менеджер паролів** - Система управління паролями в пам'яті
- **Інтеграція сховища** - Функції saveCustomers/loadCustomers з інтеграцією шифрування

### 📚 Мовна Підтримка
- **13 мов оновлено** - Нові ключі перекладу шифрування у всіх мовних файлах
- **Нові ключі**: `encryption.enableEncryption`, `encryption.useSamePasswordAsApp`, `encryption.encryptionPassword`, `encryption.passwordMinLength`

## v1.6.0 (2025) - 📊 Інтерактивні віджети & налаштування продуктивності великих таблиць

### 🧠 Інтерактивні графіки та детальні модальні перегляди
- **Основні графіки панелі використовують уніфікований компонент `InteractiveChart`** з клікабельними точками даних та анімованим детальним модальним переглядом
- **Підказка та детальний перегляд локалізовані**, показуючи читабельні мітки (дохід, витрати, чистий прибуток, кількість пропозицій)
- **Період часу можна встановити безпосередньо з графіка тренду** (тижневий / місячний / річний) використовуючи кисть, нарізані дані потрапляють до Home → Dashboard

### 🧵 Віртуальна прокрутка для великих списків
- **Власна віртуальна прокрутка** для списку пропозицій та таблиці філаментів – відображаються лише видимі рядки, забезпечуючи плавну прокрутку навіть з 10k+ записами
- **Налаштування → Бібліотека філаментів** використовує той самий шаблон, підтримуючи повну палітру 12,000+ кольорів адаптивною
- **Позиція/висота смуги прокрутки залишається правильною** завдяки елементам відступу вище та нижче видимого діапазону

### 📋 Розширене сортування та фільтрація таблиць
- **Багатоколонкове сортування** на сторінках Філаментів та Пропозицій (клік: зростання/спадання, Shift+клік: побудова ланцюжка сортування – напр. "Марка ↑, потім Ціна/кг ↓")
- **Налаштування сортування збережені в `settings`**, тому переважний порядок зберігається після перезапуску
- **Філаменти**: фільтри на рівні колонки для марки, матеріалу/типу та значення кольору/HEX
- **Пропозиції**: фільтр суми зі значеннями мін/макс та фільтри діапазону дат (від / до)
