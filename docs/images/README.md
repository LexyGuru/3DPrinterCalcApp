# Images Directory

This directory contains images used in the README files.

## Structure

- `icon.png` - Main application icon (512x512)
- `Square310x310Logo.png` - Application logo for documentation

## Adding New Images

When adding new screenshots or images to the README:

1. Place the image file in this directory (`docs/images/`)
2. Reference it in the README using: `docs/images/your-image.png`
3. Use relative paths so images work on GitHub

## Example Usage

```markdown
![Description](docs/images/your-image.png)
```
